# cub3D
10th project for 42cursus

## 機能
- 入力：シーンの.cubファイル一つ
  - 東西南北で別のタイル
  - 床と天井の色
- WASDで移動
- 矢印キーで視点移動
- ESCで終了

## Algorithm
- Raycasting
	- Digital Differential Analyzer (DDA)
		- [Bresenham's line algorithm](https://en.wikipedia.org/wiki/Bresenham's_line_algorithm)
	- Texture mapping
	- collision detection
